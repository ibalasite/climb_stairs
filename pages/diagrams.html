03

Data Flow

Join room + start game sequence

click to enlarge
sequenceDiagram
    participant C as Client
    participant F as Fastify Pod
    participant R as Redis

    C->>F: POST /api/v1/rooms/join
    F->>R: SETNX room:{code}
    R-->>F: OK
    F-->>C: 201 { roomCode, sessionToken }

    C->>F: WSS /ws?token=JWT
    F->>F: Verify JWT
    F-->>C: ROOM_STATE_FULL

    C->>F: WS: START_GAME
    F->>R: WATCH + MULTI/EXEC
    F->>F: GenerateLadder(seed,N)
    F->>R: SET ladder
    F->>R: PUBLISH events
    R-->>F: Pub/Sub deliver
    F-->>C: ROOM_STATE {running}

08

API Flow

Reveal sequence from host trigger to client animation

click to enlarge
sequenceDiagram
    participant H as Host Client
    participant F as Fastify Pod
    participant R as Redis
    participant P as Player Clients

    H->>F: WS: REVEAL_NEXT
    F->>R: INCR revealedCount
    R-->>F: count = 1
    F->>R: PUBLISH events REVEAL_INDEX
    R-->>F: Pub/Sub → all pods
    F-->>H: REVEAL_INDEX {index:0, playerId}
    F-->>P: REVEAL_INDEX {index:0, playerId}
    Note over H,P: All clients animate path 0

    H->>F: WS: REVEAL_ALL_TRIGGER
    F->>R: GET ladder results
    R-->>F: ResultSlot[]
    F->>R: PUBLISH REVEAL_ALL
    F-->>H: REVEAL_ALL {results}
    F-->>P: REVEAL_ALL {results}
    Note over H,P: All animate remaining paths
    F-->>H: ROOM_STATE {status:finished}
    F-->>P: ROOM_STATE {status:finished}
