Version v2.0 — Draft — Ladder Room Online MVP v1

1. 產品概述

Ladder Room Online 是一款基於 LINE 爬樓梯玩法的 HTML5 線上多人互動遊戲。玩家可從不同地點使用瀏覽器進入同一個 Room，等待主持人開始。主持人設定中獎名額後，系統隨機生成樓梯，所有玩家沿各自起點向下走，最終對應到底部的中獎或未中獎結果。

一句話定義：一款以 LINE 爬樓梯規則為基礎，支援異地玩家加入同一 Room，由主持人控制開局與揭曉的 HTML5 線上多人抽獎遊戲。

2. 產品背景

2.1 現有問題

LINE 爬樓梯遊戲是常見的抽獎互動形式，但原生版本存在以下限制：

問題說明
人數限制超過一定人數後只顯示結果，不顯示完整走線過程
視覺不完整人數多時畫面顯示不完整
規則僵化難以自訂房間機制與獎項規則
無法獨立部署不適合作為獨立品牌活動頁或大型線上互動玩法

2.2 為何現在做

線上活動、尾牙、直播互動需求日增，需要一個可獨立部署、支援異地多人的解決方案。

3. 產品目標

3.1 核心目標

  • 玩家可透過 Room Code / 連結加入房間
  • 主持人可建立房間並控制開局
  • 主持人可設定中獎名額
  • 開始後產生唯一且固定的樓梯地圖
  • 所有玩家共享同一組樓梯結構與結果邏輯
  • 每位玩家可看到自己的走線與最終結果

4. 產品定位

定位：線上房間制互動抽獎遊戲，屬於「活動遊戲工具」而非競技遊戲。

定位關鍵字：Party Game、Lottery Game、Event Interaction Tool、Room-based Mini Game

維度Ladder Room OnlineLINE 爬樓梯
人數上限50 人~10 人以內完整動畫
異地多人完整支援僅限同裝置群組
結果可驗證PRNG seed 公開不透明
自訂中獎名額完整支援固定格式
獨立部署自建或 SaaS無法

5. 角色定義

Host（主持人）

負責建立房間、設定中獎名額、啟動遊戲、控制揭曉節奏、踢除玩家。通常是活動主辦人、老師、或直播主。技術能力中等，主要透過 UI 操作。

Player（玩家）

掃描 QR Code 或輸入 Room Code 加入房間。輸入暱稱後等待主持人開局。希望看到自己的路徑動畫與最終結果。技術能力不要求。

Spectator（旁觀者）

（Post-MVP）不參與抽獎，僅觀看遊戲進行。適合直播場景。

6. 核心流程

  1. 建立房間：主持人點擊「建立房間」→ 取得 6 碼 Room Code
  2. 玩家加入：玩家輸入 Room Code + 暱稱 → 進入等待畫面
  3. 設定中獎名額：主持人設定 W（1 ≤ W ≤ N-1）
  4. 開始遊戲：主持人點擊「開始」→ 系統生成樓梯，狀態變 running
  5. 揭曉：逐步（手動/自動）或一鍵揭曉所有路徑
  6. 結果顯示：所有玩家看到最終結果，狀態變 finished
  7. 再玩一局：主持人可 reset 房間，重複流程

8. UX 設計原則

  • 低門檻：玩家加入只需輸入暱稱和房間碼，無需帳號
  • 即時回饋：所有狀態變更立即同步至所有端點
  • 懸念感：逐步揭曉動畫製造緊張氛圍
  • 公平透明：PRNG seed 可公開驗證，結果不可篡改
  • 跨裝置：桌機與手機均可流暢使用，Canvas FPS 達標
