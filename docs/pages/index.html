Ladder Room Online 文件中心

基於 LINE 爬樓梯玩法的 HTML5 線上多人互動抽獎遊戲。玩家從不同地點使用瀏覽器進入同一個 Room，主持人建立房間並掌控揭曉節奏，Mulberry32 PRNG 算法確保每局結果在所有客戶端 100% 一致。

技術棧

Runtime Node.js 20 LTS
語言 TypeScript strict mode，前後端共用
HTTP Server Fastify REST API /api/*
WebSocket ws (原生) maxPayload: 65536
狀態 / 快取 Redis 原子操作 + Pub/Sub
前端 Vanilla TS + Vite HTML5 Canvas，零框架
測試 Vitest + Playwright Unit / Integration / E2E
CI/CD GitHub Actions lint → test → build → deploy
